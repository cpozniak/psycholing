Description de l'étude

Vous verrez apparaître une suite de lettres pour laquelle vous devrez décider le plus rapidement et le plus exactement possible s’il s’agit d’un mot qui existe dans la langue française ou non. Pour ce faire, vous devez appuyer sur la touche D de votre clavier si cette suite de lettres n'existe pas ou sur la touche K de votre clavier si elle existe.

Vous ne nous fournissez aucune donnée permettant de vous identifier en participant à cette étude.
